# ADR 索引

ADR 是 `Architecture Decision Record`，用來記錄重要技術決策。

它不是單純的說明文件，而是回答：

- 我們為什麼做這個選擇
- 我們評估過哪些替代方案
- 這個決策的代價是什麼
