Enterprise Docs Hub 為微服務範本準備的 GitHub Pages 文件入口
GitHub Pages 文件站展示

三套可直接部署的靜態文件網站

這裡提供 3 種風格的文件閱讀介面，全部都是純 HTML、CSS、JavaScript，不需要 Node.js，不需要 build，直接放到 GitHub Pages 就可以用。

Classic Docs

最像正式文件站。左側導覽、右側內容、搜尋與目錄都在。適合團隊長期使用，也最接近企業內部知識庫風格。

打開這版 直接看快速開始

Cards Explorer

用卡片探索整份文件，視覺輕快，初學者比較不容易被大量文字嚇到。適合先找主題，再進入細節。

打開這版 直接看開發手冊

Storybook Guide

把文件整理成「我是誰，我該先看哪裡」的故事式入口。適合新人、學生、第一次接觸微服務的人。

打開這版 直接看角色導覽

部署方式

本專案已額外提供 GitHub Pages workflow。只要推到 GitHub，啟用 Pages 使用 GitHub Actions 即可。

  • 入口頁：`/`
  • Classic Docs：`/classic/`
  • Cards Explorer：`/cards/`
  • Storybook Guide：`/storybook/`
